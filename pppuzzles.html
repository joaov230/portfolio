Sobre o projeto

Motivação:

O projeto Python Parsons Puzzles surgiu como uma forma de incentivar e ajudar no aprendizado de programação para iniciantes, utilizando a linguagem Python e Problemas de Parsons, exercícios parecidos com Programação por Blocos, como amplamente divulgado pelo Hour Of Code, porém já utilizando a sintaxe de Python.

Até o momento do meu TCC, não havia (ao meu conhecimento) nenhuma plataforma que utilizasse Problemas de Parsons e salvasse o progresso do usuário, e foi isso que foi criado.

Decisões técnicas:

Foram utilizadas formas inteiramente gratuitas de hospedar a plataforma. A API foi desenvolvida utilizando Google Apps Script, vinculada à tabelas no Google Sheets para armazenar os dados, como exercícios, progresso de usuário, etc, enquanto para o login foi utilizado o Google OAuth 2.0, API oficial da Google para realizar login com o Google.
